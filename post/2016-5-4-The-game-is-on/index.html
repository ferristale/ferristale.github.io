The Game is On

2016年5月4日
| 杂文
| 阅读
文章目录

跑完步回來，不想幹活，於是過來寫文（多麼光明正大的理由） （事實上從決定寫文到現在已經25分鐘..主要是回了一下關於內核問題的帖子）

從五一開始講吧

五一放假，原本是到2號，三號早上有一節課。不過考慮到30號晚上才回去，那個課也不是特別重要，而且正好一個朋友也回去了，於是決定推遲一天回去。2號那天跟朋友坐車去教倫，然後…..走回來

不得不說真的很累啊==我一個男的都累得夠嗆（當然可能是太久沒運動了），難爲人家女生了。。Anyway總算是走到家了。。那個女生是大一時候認識的，是個好女孩，有機會也許我會寫寫吧，不過其實也沒啥可寫的，都沒啥故事啊==

回家三天，第一天似乎是維護家裏電腦，並繼續了合併郵箱的進度，晚上開始看Coursera，第二天一早繼續看Coursera，到最後一小時才終於把那一週的課程看完。不得不說全英課程看得真累..看完了，下午跟朋友出去，晚上回來打卡。第三天開始看論文，慚愧項目過了那麼久了我才開始看論文，真的感謝隊友大腿。

路況不好，回到學校已經5點半，吃個飯，打卡，開始找實驗相關資料，第三天過去。咦不知不覺就到了今天，把資料總結發給老師，咦，就沒了。其實是玩了幾個小時的刺客信條

這幾天裏有一個貫穿的，就是折騰了好多邊CDN

合併郵箱

回家前一天爲被衆多郵箱感到深深的苦惱，終於決定處理他們。怎麼處理呢，上了一下知乎。最後決定把騰訊域名郵箱註銷，想着說註冊到騰訊的企業郵箱，但是因爲要等24小時，遂又扒了一下文，最終把域名郵箱放在了zoho(疼訊啊疼訊，你的24小時讓你失去了一個客戶啊～雖然不是付費的~~)。接着把各種郵箱pop到Gmail。不得不說Gmail好專業，好多功能我看了超久還是不太明白，反正都合併到Gmail就對了

現在還剩下一直用的yeah.net和qq郵箱，yeah.net是我一直用的，基本上網上的帳號都綁定了他，也造成了現在廣告纏身，而且特麼都一個形式，網易卻不支持內文過濾規則，見了鬼了。最離譜的是同一個地址不同的時間發了好幾封給我，我就納悶居然有光明正大不換地址的垃圾郵件！網易居然沒攔掉！容我最後BB一句，看着郵箱裏的一堆垃圾郵箱，在看看垃圾郵件文件夾裏寥寥幾封以及頂部的廣告：網易郵箱已爲你攔截超過99.98%的垃圾郵件。。。。

不想把他也合併過去，就是因爲郵件太多，煩不勝煩，一直對他很有感情的，可是近來逐漸降溫，看怎樣吧。至於QQ郵箱，不知道那一堆羣郵件會不會也pop過去啊，一堆垃圾。。

總之，現在已經把常用的郵箱pop到Gmail，（說得好聽其實常用都沒有處理..）驚喜的是，在我的安卓手機上，設置接收Gmail，在不翻牆的情況下居然也能收件！！太贊啊！估計是Gmail的IMTP服務器沒有被牆吧，唯一遺憾的是不能用Gmail上連接的其他郵箱發件。而且測試發件也是成功的！不能更讚啊有木有！！！ (By the way,試了一下yeah.net的pop，居然設置不了，不知道問題出在哪裏 。再說吧)

The Game Is On

這是夏洛克裏面夏洛克的一句，口頭禪吧。The Game Is On.對我來說，是的

大學快三年了，三年裏幾乎都是玩的，第一年忙着社團活動，倒是沒掛科。第二年，競選的失敗給了我很大打擊，最傷人的並不是競選失利，而是評委的咄咄逼人，以及對我的各種否認，也許，我真的沒什麼成績？真的付出不夠。去另一個協會開會的那晚，在自行車上狠狠哭了一次，我以爲沒事了。現在才發現，那件事傷我最深。從那時起，不再上進，不再努力，不再積極。而我居然一直不覺。 那年，掛了四科，給現在的自己埋下沉重的包袱。不管怎麼講最後我決定考研。成績不好，掛過科，但那只代表過去的我，不能證明現在的我，更不能決定未來的我，不是嗎。既然輸了開頭，那就在最後發力，現在努力還不晚。 今天（今天是5號了），下午去了實驗室，正式接觸到了未來一年半將要進行的項目，接下來的時間裏，可能沒事就要往實驗室跑。另一方面聯繫了植物園，暑假要去那邊實習。其實完全可以在植物園實習的時候順帶把論文做了，那就沒必要再花可能更多的時間來做另一個項目，到時候論文只能選其一。我也糾結了很久，最後是有人來找我，好那就做吧，論文是其次，主要是培養自己科研的能力，雖然不考本專業，但最研究的能力多點總是沒錯的。加之暑假本來就是要去實習的，區別只是學校安排的是一個半月，我的是兩個月。也是一種經歷，對吧。講白了就是優勢，當然這裏面所花的時間代價就加大了本來就成績不好的劣勢罷了。 Anyway,The Game IS On

小劇場

Assassin’s Creed

我居然拼對了…..好吧，在這種時候，我又開始了我的一代刺客信條。過了這麼多年，最喜歡的遊戲還是它，一直想玩的一代也依然還沒玩。箇中原因，一是以前都在玩lol，二是每每玩到一半（其實沒一半啦），重裝系統，存檔又沒了，搞得我都不太想去碰它。這次直接是給他做了個軟連接，哼哼（傲嬌臉0.0）

堅果雲-別用慣了免費就把收費當流氓

最近幾家國內網盤倒閉，網民人人自危，都在尋找好的靠譜替代品。於是不少用戶轉到堅果雲，然而當他們看到免費只有一個G，收費是每月20的時候，各種罵娘。太貴的聲音不絕於耳。

我想說的是，別用慣了免費就把收費當流氓

在收費用戶眼裏，你們這種要求纔是耍流氓！

CDN

這幾天把CDN換了又換，CLoudflare那個節點，比直連還慢，incapsula那個日本節點早被玩壞，香港也是，美國。。特麼我還不如直連。（不過哪怕PIng高了，但速度好像確實比直連快） 其實，Hexo最大的優勢就是全靜態，跟CDN那是絕配，就算沒CDN，也要比WP快，但可能因爲我是美國主機，加上換了Hexo後一是找不到利用七牛的方法，速度上反而比之前WP加七牛要慢了。特別是第一次加載，真的是。。。受不了啊。。如果能用七牛CDN，全靜態絕對爽歪歪。可是用不了啊。國外的CDN用的我都要抓牆角。醉。。 不過話說回來，現在這個要加載的文件比之前WP的時候多多了。拖速度。我再想是不是換個主題，雖然這個不錯，但速度太慢。

不寫了，有空再寫別的吧

話說，360的字體库也很拖速度啊，什麼鬼啊！比谷歌字體還慢！
