VESTA面板升級PHP7.0

2015年12月14日
 | 瞎搞
文章目錄

 

VESTA的各種部件安裝位置令人神傷

網上爬了很多文章

終於無痛升級PHP7.0

 

修改/etc/yum.repos.d/remi-php70.repo

cd /etc/yum.repos.d/
vi remi-php70.repo

把[remi-php70]下的enabled=0

改為enabled=1

如果沒有這個文件，那你可能要更新一下

yum install epel-release
yum install http://rpms.famillecollet.com/enterprise/remi-release-6.rpm

如果要安裝5.5或5.6，就修改remi.repo文件的相應數值

即：

cd /etc/yum.repos.d/
vi remi.repo

vi编辑器的一些命令：i 是编辑；ESC 键是退出编辑；:wq!+回车是保存并退出；:q! 是不做修改直接退出

開始升級

執行命令

yum update php --enablerepo=remi -y

完成之後會有Complete的字樣

重啟服務器

service nginx restart
service vesta restart

Enjoy it !
