近期博客变动

2023年1月7日
 | 瞎搞

 

博客很久没有更新了，因为新年，有些东西想写，所以更新下，顺便更新了主题，有了些变化：

  • Bugs
    • 现在默认新页面打开链接了，不知道为啥，不会修
    • 主题更新引入了国际化语言支持，但 categories，tags 怎么改都是英文，不知道为啥
    • 点进 categories，tags，会有些奇怪的数字出现，不会修
    • 不支持 iframe 了，以前插在文章里的歌、视频不能显示了
  • Fixs
    • 修改了语言设置，可以部分显示中文
    • 增加了网易云和 bilibili 的 shortcode，以另一种方式插歌（随时失效）
    • 开启图片点击放大
    • 删除无效 css & js 引用
    • 修复一些未设置 链接 的文章
  • Adjust
    • 持续删除过于私人的文章
    • 调整 tags
