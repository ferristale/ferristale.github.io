比如說服務器啓動rsync有兩種方法
比如說Hexo的插件似乎並非支持全部方法

VPS安装nginx

使用Hexo，VPS上只要安装nginx和git就足够了，大大地节省了空间
而nginx的安装，我找到了三种方法
一个是用编译，一个是启用rhel
这两种一个可以自定义，一个算是经典安装吧

但是我找到了一种最小安装的（至少比前2小很多）
不足的是这个安装完好像没有gzip插件，不过可以手动安装

yum install nginx
