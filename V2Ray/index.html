Multi-V2Ray 一键部署

2019年7月28日
 | 瞎搞

安装

source <(curl -sL https://git.io/fNgqx) --zh

脚本更新

source <(curl -sL https://git.io/fNgqx) -k

师妹想科学上网，我就，重新刷了一下V2Ray

首先是脚本，在谷歌上输入VaRray 一键，出来的都是233的脚本 本文用的是multi-v2ray脚本，是在233的github Issue里发现的 用法的原因嘛，，因为看起来更新更勤一点

其实功能都差不多，该有的都有。这里说一下注意的几个问题

  1. 脚本自动安装，并自动设置端口、加密方式、id等，总之就是开箱即用，对配置不满意可以直接输入V2Ray修改
  2. 动态端口默认关闭
  3. V2Ray core默认自动更新
  4. 时区自动校正
  5. 客户端建议使用V2RayN，可链接导入  

还有就是，kcp、BBR等东西慎用吧，在不确定小鸡会不会被BAN的情况下 端口容易被封就用范围端口 注意范围端口不等于动态端口
